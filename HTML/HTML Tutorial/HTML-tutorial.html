Article:

An Article specifies independent, self-contianed content. An article should make sense on its own and it should be possible to distribute it independently from the rest of the site.

Aside:

An Aside defines some content aside from the content it is placed in. It should be related to the surrounding content.

Blockquote:

A Blockquote defines a long quotation, and/or specifies a section that is quoted from another source. (Browsers usually indent)

Button Tag:

Defines a clickable button; Can put content (like text or images) inside this element.

Input Tag - Button:

Specifies an input field where the user can enter data; Input Fields vary depending on the 'type' attribute &
is used within a Form Element to declare input controls that allow users to input data.

Form Tag:

Used to create an HTML form for user input; Can contain one or more of the following elements:
input, textarea, button, select, option, optgroup, fieldset, label.

Table:

Defines an HTML Table; consists of the elements: table, tr (Table Row), th (Table Header), td (Table Cell)
Complex Tables may include elements: caption, col, colgroup, thead, tfoot, tbody
